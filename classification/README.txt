# Sentiment Analysis of Tweets

This project demonstrates sentiment analysis of tweets using a machine learning model trained on labeled tweet data. It utilizes Django as the web framework for creating a simple web application.

## Setup

To set up the project, follow these steps:

1. Clone the repository.
2. Install the required dependencies using `pip install -r requirements.txt`.
3. Ensure that Python 3.x and Django are installed on your system.
4. Move the `vectorizer.pkl` and `model.pkl` files to the `tweets` directory.

## Usage

To run the web application locally, execute the following command:

```bash
python manage.py runserver
